# 🧼 JCF Productos de Limpieza

Aplicación web simple para visualizar productos de limpieza, agregarlos a un carrito y enviar pedidos directamente por WhatsApp.

---

## 🚀 Objetivo

Facilitar la venta de productos de limpieza para clientes comunes, automatizando pedidos sin necesidad de responder manualmente cada mensaje.

---

## 🧩 Funcionalidades actuales

* 📦 Lista de productos
* ➕ Agregar productos al carrito
* 🛒 Visualización del carrito
* 📲 Envío del pedido por WhatsApp

---

## 🛠️ Tecnologías utilizadas

* HTML
* CSS
* JavaScript (Vanilla)

```

```

2. Abrir el archivo:

```
index.html
```

3. Usar la app en el navegador

---

## 📲 Funcionamiento del pedido

El usuario:

1. Selecciona productos
2. Se agregan al carrito
3. Hace clic en "Enviar por WhatsApp"
4. Se abre WhatsApp con el pedido ya escrito
